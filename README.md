# 🌸 Telegram-бот «Цветочный магазин»

Полнофункциональный Telegram-бот для онлайн-магазина цветов на Python 3.11+, aiogram 3.x, SQLAlchemy async, SQLite.

## Возможности

- **Пользователи:** регистрация, каталог букетов, корзина, оформление заказа, оплата по скриншоту
- **Админы:** управление товарами, просмотр заказов и пользователей, уведомления о новых заказах и оплатах

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

1. Скопируйте `.env.example` в `.env`
2. Укажите `BOT_TOKEN` (получить у [@BotFather](https://t.me/BotFather))
3. Укажите `ADMIN_IDS` — Telegram ID администраторов через запятую (узнать у [@userinfobot](https://t.me/userinfobot))

## Запуск

```bash
python bot.py
```

## Структура проекта

```
bot.py              # Точка входа
config.py           # Конфигурация
database/           # Модели и БД
handlers/           # Обработчики (user, admin)
keyboards/          # Клавиатуры
services/           # Бизнес-логика
states/             # FSM-состояния
utils/              # Утилиты (уведомления)
```

## Номер телефона для оплаты

В `handlers/user/order.py` в тексте инструкции по оплате замените `<вставьте номер телефона для оплаты здесь>` на реальный номер.
